# Original Sling Physics Game

## Project Documentation

### Overview
The Original Sling Physics Game is a physics-based puzzle game where players use a slingshot to launch projectiles to hit targets and solve challenges.

### Features
- **Engaging gameplay**: Players can enjoy various levels that challenge their physics knowledge and reflexes.
- **Physics engine**: Utilizes a realistic physics engine to simulate projectile motion.
- **Intuitive controls**: Simple touch controls make it easy for players of all ages.

### Getting Started
To set up the game locally:
1. Clone the repository:
   ```
   git clone https://github.com/boburbeklv/original-sling-physics-game.git
   ```
2. Navigate to the project directory:
   ```
   cd original-sling-physics-game
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Run the game:
   ```
   npm start
   ```
